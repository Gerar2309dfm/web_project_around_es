# Around The U.S.

## Descripción del proyecto
Around The U.S. es una aplicación web interactiva que permite a los usuarios crear y gestionar una galería de tarjetas con lugares de interés. Los usuarios pueden editar su información de perfil, agregar nuevas tarjetas con imágenes y nombres de lugares, dar "me gusta" a las tarjetas, eliminar tarjetas y ver las imágenes en tamaño completo.

## Funcionalidad
- Editar información del perfil (nombre y descripción)
- Agregar nuevas tarjetas con imagen y título
- Dar "me gusta" a las tarjetas
- Eliminar tarjetas
- Ver imágenes en popup de tamaño completo
- Validación de formularios en tiempo real
- Interfaz responsive para diferentes dispositivos

## Tecnologías y técnicas utilizadas
- **HTML5**: Estructura semántica del contenido
- **CSS3**: Estilos y diseño responsive con metodología BEM
- **JavaScript ES6**: Programación orientada a objetos con clases
- **Módulos ES6**: Organización del código en componentes reutilizables
- **Webpack**: Empaquetado y optimización de archivos
- **Git**: Control de versiones
